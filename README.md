# 🌎 Projeto de Dados | Desmatamento na Amazônia

## 📌 Sobre o Projeto
Pipeline de dados desenvolvido para analisar informações públicas sobre desmatamento na Amazônia, desde a coleta até a visualização interativa.

O projeto teve como foco praticar análise de dados, ETL, cloud e construção de dashboards utilizando Python, AWS e Streamlit.

---

# 🎯 Objetivos
- Coletar dados do Kaggle
- Limpar e transformar dados com Python + Pandas
- Criar métricas analíticas
- Armazenar dados na AWS S3
- Consultar dados com SQL / Athena
- Construir dashboard interativo no Streamlit

---

# 🛠️ Tecnologias Utilizadas
